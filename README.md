# GardenLawn Hyvatheme Documentation

This document provides an overview of the GardenLawn Hyvatheme.

## Directory Structure

- **Amasty_GoogleMap**: Customizations for Amasty Google Map module.
- **Amasty_SeoHtmlSitemap**: Customizations for Amasty SEO HTML Sitemap module.
- **Hyva_Checkout**: Customizations for Hyva Checkout.
- **Hyva_SmileElasticsuite**: Customizations for Hyva Smile ElasticSuite integration.
- **Hyva_Theme**: Core Hyva Theme customizations.
- **Magento_Catalog**: Customizations for Magento Catalog module.
- **Magento_Checkout**: Customizations for Magento Checkout module.
- **Magento_Cms**: Customizations for Magento CMS module.
- **Magento_Contact**: Customizations for Magento Contact module.
- **Magento_Customer**: Customizations for Magento Customer module.
- **Magento_Directory**: Customizations for Magento Directory module.
- **Magento_Email**: Customizations for Magento Email templates.
- **Magento_InventoryInStorePickupSales**: Customizations for In-Store Pickup Sales.
- **Magento_LayeredNavigation**: Customizations for Layered Navigation.
- **Magento_LoginAsCustomerFrontendUi**: Customizations for Login as Customer UI.
- **Magento_Newsletter**: Customizations for Newsletter module.
- **Magento_OrderCancellation**: Customizations for Order Cancellation.
- **Magento_ProductAlert**: Customizations for Product Alert module.
- **Magento_ReCaptchaFrontendUi**: Customizations for ReCaptcha UI.
- **Magento_Review**: Customizations for Product Reviews.
- **Magento_Sales**: Customizations for Sales module.
- **Magento_SendFriend**: Customizations for Send to Friend module.
- **Magento_Store**: Customizations for Store module.
- **Magento_Swatches**: Customizations for Swatches.
- **Magento_Tax**: Customizations for Tax module.
- **Magento_Theme**: General theme customizations.
- **Magento_Wishlist**: Customizations for Wishlist module.
- **etc**: Configuration files.
- **i18n**: Translation files.
- **view**: View files.
- **web**: Web assets (CSS, JS, images).

## Overview

This is the main frontend theme for the GardenLawn project, based on the Hyvä Theme. It includes customizations for various Magento modules and Hyvä-specific extensions to provide a performant and tailored user experience.
